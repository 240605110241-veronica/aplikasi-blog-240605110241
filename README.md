# aplikasi-blog-240605110241
RuangCerita merupakan aplikasi blog full-stack berbasis Laravel yang dikembangkan sebagai proyek pemenuhan Ujian Akhir Semester mahasiswa Teknik Informatika UIN Maulana Malik Ibrahim Malang. Aplikasi ini menghadirkan sistem Content Management System (CMS) dinamis yang dibalut dengan antarmuka bergaya Soft Minimalist dan Aesthetic Editorial Magazine menggunakan Tailwind CSS. Platform ini difokuskan sebagai ruang publikasi digital yang mengedepankan pengalaman membaca (reading experience) tanpa distraksi, merangkul berbagai kategori kreatif dan personal. Dengan perpaduan palet warna bernuansa hangat serta tata letak grid yang responsif, RuangCerita menjadi medium berekspresi yang elegan, interaktif, dan modern.
